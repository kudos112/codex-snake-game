# codex-snake-game

A minimal classic Snake game built with vanilla HTML, CSS, and JavaScript.

## Features

- Grid-based snake movement
- Food spawning and snake growth
- Score tracking
- Game-over on wall and self collision
- Pause and restart controls
- Keyboard controls (Arrow keys / WASD) and on-screen buttons

## Run locally

```bash
npm run dev
```

Open `http://localhost:5173`.

## Test

```bash
npm test
```

## Controls

- Move: `Arrow keys` or `W/A/S/D`
- Pause/Resume: `Space` or `Pause` button
- Restart: `R` or `Restart` button
